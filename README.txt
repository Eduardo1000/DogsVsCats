Repositório para aprender Deep Learning. 
Primeiro projeto de Deep Learning realizado por mim. 
O projeto é retirado do Kaggle e se baseia em confeccionar uma rede neural para classificação de Gatos e Cachorros. 
O dataset pode ser baixado diretamente do Kaggle(https://www.kaggle.com/c/dogs-vs-cats/data) e contém cerca de 37.500 imagens sendo 25.000 para treino/validacao e 12.500 para test.

Para utilizar basta apenas baixar: 
- Python 3.6
- pip3 install tensorflow, numpy, opencv-python
